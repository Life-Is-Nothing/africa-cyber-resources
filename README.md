## 🏆 Certifications recommandées (par ordre)

```
Niveau Débutant:
├── ISC2 CC (gratuite) ⭐⭐⭐⭐⭐
├── CompTIA Security+
└── CSOCF — CertNexus

Niveau Intermédiaire:
├── CEH — EC-Council
├── eJPT — eLearnSecurity  
└── CompTIA PenTest+

Niveau Avancé:
├── OSCP — Offensive Security 💀
├── CISSP — ISC2
└── CISM — ISACA
```

## 🔧 Outils essentiels (tous gratuits)

### Pentest & Reconnaissance
```
Nmap          — Scanner réseau
Nikto         — Scanner web
Gobuster      — Brute-force répertoires
theHarvester  — OSINT & reconnaissance
Maltego CE    — Visualisation OSINT
Shodan        — Moteur de recherche IoT
```

### Défense & Monitoring
```
Wazuh         — SIEM open source (⭐ recommandé)
Suricata      — IDS/IPS réseau
Zeek          — Analyse réseau
OSSEC         — HIDS
Graylog       — Gestion des logs
```

### Forensics & Analyse
```
Autopsy       — Forensics disque
Volatility    — Analyse mémoire
Wireshark     — Analyse réseau
CyberChef     — Décodage/encodage
Ghidra        — Reverse engineering (NSA)
```

## 🗺️ Roadmap suggérée (12 mois)

```
Mois 1-2:   Fondamentaux réseau (TCP/IP, OSI, DNS, HTTP)
Mois 3-4:   Linux administration & scripting Bash
Mois 5-6:   ISC2 CC + TryHackMe parcours débutant
Mois 7-8:   Pentest intro (Metasploit, Nmap, Burp Suite)
Mois 9-10:  CTF réguliers (root-me, picoCTF)
Mois 11-12: eJPT certification + portfolio GitHub
```
